Что такое JavaScript?

JavaScript — это язык, который позволяет Вам применять сложные вещи на web странице — каждый раз, когда на web странице происходит что-то большее, чем просто её статичное отображение, отображение периодически обновляемого контента, интерактивных карт, анимация 2D/3D графики, прокрутка видео в проигрывателе, и т.д. — можете быть уверены, что, скорее всего, не обошлось без JavaScript Это третий слой слоёного пирога стандартных web технологий, позволяющий создать динамически обновляемый контент, управлять мультимедиа, анимировать изображения и делать много прочих вещей
К меню

Ядро языка JavaScript состоит из обычных возможностей, которые позволяют делать следующее:

  • Хранить данные внутри переменных
  • Производить различные операции со строками
  • Запускать код в соответствии с определенными событиями, происходящими на web странице. Например, можно использовать CLICK событие для определения момента, когда кнопка была нажата, чтобы в соответствии с этим запускался определенный код
Еще более увлекательным является функциональность, созданная поверх основного языка JavaScript. Так называемые интерфейсы прикладного программирования (API) предоставляют вам дополнительные способности для использования в вашем коде JavaScript. API - это готовые наборы блоков кода, позволяющие разработчику реализовывать программы, которые в противном случае было бы трудно или невозможно реализовать.
